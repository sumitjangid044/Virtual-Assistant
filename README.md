# 🤖 Virtual Assistant

A full-stack AI-powered Virtual Assistant that can understand voice commands and perform tasks like searching Google, YouTube, checking weather, and more.

---

## 🚀 Live Demo

- 🌐 Frontend: https://virtual-assistant-qz5f.onrender.com  
- ⚙️ Backend: https://virtualassistant-backend-raw2.onrender.com  

---

## ✨ Features

- 🎤 Voice Recognition (Speech-to-Text)
- 🔊 Text-to-Speech (Assistant speaks)
- 🔍 Google Search
- ▶️ YouTube Search & Play
- 📱 Open Apps (Instagram, Facebook, Calculator)
- 🌦️ Weather Search
- 👤 User Authentication (Signup/Login)
- 🧠 AI-powered responses (OpenAI API)

---

## 🛠️ Tech Stack

### Frontend:
- React.js
- Tailwind CSS

### Backend:
- Node.js
- Express.js
- MongoDB

### Other:
- OpenAI API
- SpeechRecognition API
- SpeechSynthesis API

### 1. Clone Repo

cd Virtual-Assistant


### 2. Backend
cd backend
npm install
npm run dev

### 3. Frontend
cd frontend
npm install
npm run dev
